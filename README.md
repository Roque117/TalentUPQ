# TalentUPQ 🎓💼

**TalentUPQ** es una aplicación web de vanguardia diseñada para optimizar y fortalecer la vinculación laboral entre los estudiantes/egresados de la **Universidad Politécnica de Querétaro (UPQ)** y el sector productivo.

## 🚀 Propósito del Proyecto
El objetivo primordial es ofrecer una herramienta confiable y exclusiva que facilite la búsqueda de empleo para la comunidad universitaria y la publicación de vacantes para empresas, buscando impactar al menos al **70% de la matrícula** en su primer año de operación.

## ✨ Funcionalidades Principales
El sistema cuenta con tres perfiles de acceso (Estudiante, Empresa, Administrador) y ofrece:

*   **Gestión de Talento:** Registro de usuarios, creación de perfiles profesionales y carga de CV (PDF/DOCX).
*   **Mercado Laboral:** Publicación de vacantes intuitiva y sistema de postulaciones centralizado.
*   **Supervisión Administrativa:** Validación de perfiles y supervisión de calidad de las vacantes por parte de la UPQ.
*   **Comunicación:** Sistema de notificaciones vía correo y plataforma sobre el estado de postulaciones.
*   **Seguridad:** Cifrado de datos sensibles y cumplimiento con la **LFPDPPP** (Protección de Datos Personales).

## 🛠️ Stack Tecnológico
Para garantizar la escalabilidad y robustez, utilizamos tecnologías modernas de nivel empresarial:

*   **Backend:** Python (Flask/Django recomendado según estructura)
*   **Base de Datos:** SQL Server
*   **Infraestructura:** Amazon Web Services (AWS)
*   **Diseño UI/UX:** Figma
*   **Gestión de Proyecto:** Metodología Scrum y ClickUp
